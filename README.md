# xxlSearch

Search for a word withing all Excel files in a folder. 

## Install
```console
$ pip install xxlSearch
```

## Quick start
```console
$ xxlSearch -h
``` 

## Example
```console
$ xxlSearch . katt


[92m========== SEARCHING . ==========[0m

```
